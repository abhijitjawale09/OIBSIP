# Weather App — OIBSIP Python Task 4

## 📌 Project Overview
A GUI-based Weather App built with Python and Tkinter.
Fetches real-time weather data using OpenWeatherMap API.

## 🚀 Features
- Modern dark-themed GUI (Tkinter)
- Current weather (temperature, humidity, wind, feels like)
- 5-Day forecast with weather emojis
- Next 6 hours hourly forecast
- Celsius / Fahrenheit toggle
- Error handling (city not found, no internet, invalid key)
- Placeholder text in search box

## 🛠️ Tech Stack
- Python 3
- Tkinter (GUI)
- Requests (API calls)
- OpenWeatherMap API (free tier)

## ⚙️ Installation & Run

### 1. Clone the repository
git clone https://github.com/YOUR_USERNAME/OIBSIP.git
cd OIBSIP/Python-Task4-WeatherApp

### 2. Install dependencies
pip install -r requirements.txt

### 3. Run the app
python main.py
