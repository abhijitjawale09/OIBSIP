# BMI Calculator — OIBSIP Python Task 2

## 📌 Project Overview
A GUI-based BMI Calculator built with Python and Tkinter.
Supports multiple users, saves history to SQLite database,
and displays a BMI trend graph using Matplotlib.

## 🚀 Features
- Modern dark-themed GUI (Tkinter)
- BMI calculation with colour-coded result
- Categories: Underweight / Normal / Overweight / Obese
- Multi-user support (enter any name)
- SQLite database to store all records
- BMI history graph (Matplotlib)
- View past records in a table
- Input validation and error handling

## 🛠️ Tech Stack
- Python 3
- Tkinter (GUI)
- SQLite3 (Database)
- Matplotlib (Graph)

## ⚙️ Installation & Run

### 1. Clone the repository
git clone https://github.com/YOUR_USERNAME/OIBSIP.git
cd OIBSIP/Python-Task2-BMICalculator

### 2. Install dependencies
pip install -r requirements.txt

### 3. Run the app
python main.py

| BMI Range     | Category     |

| 25.0 – 29.9   | Overweight   |
