# Random Password Generator — OIBSIP Python Task 3

## 📌 Project Overview
A GUI-based Random Password Generator built with Python and Tkinter.
Uses cryptographically secure generation via the secrets module.

## 🚀 Features
- Modern dark-themed GUI (Tkinter)
- Length slider (8 to 64 characters)
- Checkboxes for Uppercase, Lowercase, Numbers, Symbols
- Exclude ambiguous characters (0, O, l, 1)
- Password strength indicator (Weak / Medium / Strong / Very Strong)
- One-click Copy to Clipboard (pyperclip)
- Last 5 generated passwords history
- Cryptographically secure (secrets module)

## 🛠️ Tech Stack
- Python 3
- Tkinter (GUI)
- secrets (secure generation)
- pyperclip (clipboard)
